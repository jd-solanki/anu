## Icons

You can use `icon` prop to render icon in button.

Use `append-icon` prop to render icon after default slot.

<DemoButtonIcons />

:::details You can also use default slot to render icon.

```vue
<template>
  <ABtn>
    <i class="i-bx-star" />
    <span>Primary</span>
  </ABtn>
</template>
```
